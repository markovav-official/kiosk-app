# Deploy: один файл на целевую машину

Скопируйте **только** `flake.nix` из этой папки на целевую NixOS-машину (например в `/etc/nixos/flake.nix`). Репозиторий на целевую машину клонировать не нужно — Nix подтянет его по URL при сборке.

В файле отредактируйте:
- `inputs.kiosk-app.url` — адрес репозитория (ваш GitHub/GitLab или архив);
- параметры киоска: хостнейм, URL бэкенда, токен, Wi‑Fi, пароль разблокировки.

Затем на целевой машине:
```bash
sudo nixos-rebuild switch --flake .#kiosk
sudo reboot
```

Подробнее — в корневом [README.md](../README.md) (раздел NixOS).
